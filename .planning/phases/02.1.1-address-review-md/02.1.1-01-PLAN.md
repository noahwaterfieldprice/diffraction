3. **Create `LatticeVector` base class** with:
   - `__slots__ = ("_components", "_lattice")`
   - `__init__(self, components: Sequence[float], lattice: Lattice) -> None` — stores `np.asarray(components, dtype=np.float64)` and lattice
   - `components` property returning read-only ndarray view (`flags.writeable = False`)
   - `lattice` property returning `self._lattice` (read-only)
   - Mutable `components` setter: `self._components = np.asarray(value, dtype=np.float64)`
   - `__array__(self, dtype=None)` for numpy interop
   - `__eq__(self, other: object) -> bool` — True if same type, same lattice, components allclose
   - `__ne__` delegating to `__eq__`
   - `__repr__` showing `ClassName([x, y, z])` (components only, per CONTEXT decision)
   - `__str__` same as `__repr__`
   - `__add__(self, other)` — check `type(other) is type(self)` and same lattice, return `type(self)(self._components + other._components, self._lattice)`; return `NotImplemented` on mismatch
   - `__sub__` same pattern
   - `__neg__` returning `type(self)(-self._components, self._lattice)`
   - `__mul__(self, scalar)` and `__rmul__` for scalar multiplication — validate scalar is int/float, return `type(self)(self._components * scalar, self._lattice)`
   - `__truediv__(self, scalar)` for scalar division
   - `norm(self) -> float` using metric tensor: `sqrt(c @ lattice.metric @ c)`
   - `angle(self, other: "LatticeVector") -> float` — delegates to inner/norm

   - `inner()` with **two `@overload` signatures**:

   - `inner()` with **two `@overload` signatures** (mirror of DirectLatticeVector but reversed)
